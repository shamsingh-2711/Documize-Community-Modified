# Documize Community - Enhanced Search Module

## Project Overview

This project is a customized version of the Documize Community Edition developed as part of an internship task. The main objective was to improve the search experience by adding new UI features and enhancing the usability of the search module.

## Features Implemented

### 1. Search Filter
- Added category-based search filtering.
- Users can filter search results by:
  - All
  - Documents
  - Tags
  - Files

### 2. Search Result Pagination
- Added Previous and Next navigation buttons.
- Improved browsing experience for large numbers of search results.

### 3. Knowledge Dashboard
- Created a new Knowledge Dashboard component.
- Added dashboard template structure for future feature expansion.
